About the Pride Flag Database

The Pride Flag Database was created to help people identify pride flags and the labels related to them, as well as educate people about the 2SLGBTQIA+ community. To learn more, use the search algorithm to find a flag you want to know more about or visit the FAQ page.

About the Creator

This website was created by Emile Ohstrom (they/he) as their project for the build4good Navigators Program. This program involves creating an independent project that relates to Public Interest Technology (PIT) during the summer and attending workshops relating to design thinking, accessibility, and careers in to PIT.
